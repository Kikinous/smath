\begin{exercice}\label{exoSeconde-0052}

    Soit la fonction \( f(x)=x^2-(2x+3)^2\).
    \begin{enumerate}
 \item
 Factoriser $f(x)$ en utilisant un produit remarquable.
\item
 Calculer $f(-2)$ 
 \item
 Calculer $f(1-2/3)$
 \item
 Déterminer $x$ pour que $f(x)=0$ ( équation produit ) 
 \item
 Déterminer $x$ pour que $f(x)=-9$
            
    \end{enumerate}
