\begin{exercice}\label{exoSeconde-0052}

    Soit la fonction \( f(x)=x^2-(2x+3)^2\).
    \begin{enumerate}
        \item
 Développer et simplifier $f(x)$
 \item
 Factoriser f(x)
\item
 Calculer $f(-2)$ 
 \item
 Calculer $f(1-2/3)$
 \item
 Déterminer $x$ pour que $f(x)=0$ ( équation produit ) 
 \item
 Déterminer $x$ pour que $f(x)=-9$
            
    \end{enumerate}
Source : \cite{mVKzlq}.

\end{exercice}
