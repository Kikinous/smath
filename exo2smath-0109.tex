\begin{exercice}\label{exo2smath-0109}

Développer et réduire les expressions suivantes :
\begin{multicols}{2}
    \begin{enumerate}
        \item
$ (x + 4)(x + 3)$
\item
$ (y + 3)(2y + 8)$
\item
$ (3z + 4)(5 -6z)$
\item
 $  (- 7t + 8)(3 - 5t)$
    \end{enumerate}
\end{multicols}


\corrref{2smath-0109}
\end{exercice}
